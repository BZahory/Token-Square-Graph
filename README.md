### Installation and Starting

Clone the repo
   ```sh
   git clone https://github.com/BZahory/token-square-graph.git
   cd token-square-graph
   ```
 Install NPM packages
   ```sh
   npm install
   ```
Start the app
   ```sh
   npm start
   ```
